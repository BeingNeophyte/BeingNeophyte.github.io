Snake Game Coppercube Source

A basic Snake Game created using coppercube. In this game the player should eat the green food and avoid eating red poison. Snake will grow a bit by consuming Green food and some score will be added to player account.
while consuming a red poison will make Snake smaller and a large amount of score will be decreased. Player will die if Snake Hit the Boundry, or if Snake becomes to small.
